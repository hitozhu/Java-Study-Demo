#### 相关概念
* 缓冲区（buffer）
* 管道（channel）
* 选择器（selector）


#### 缓冲区（buffer）
> 一种基本数据类型对应一种缓冲区类型（boolean 类型除外）
* ByteBuffer
* CharBuffer
* ShortBuffer
* IntBuffer
* LongBuffer
* FloatBuffer
* DoubleBuffer

#### 管道（channel）
* FileChannel：文件
* DatagramChannel：UDP
* SocketChannel：套接字（客户端）
* ServerSocketChannel：套接字（服务端）
